# Project Images Folder

## How to Add Project Images

1. **Add your images here** - Drop screenshots, photos, or mockups into this folder
2. **Supported formats:** JPG, PNG, GIF, WebP
3. **Naming convention:** Use descriptive names like:
   - `statementpromax-dashboard.png`
   - `aws-vpc-architecture.png`
   - `covid-map-screenshot.png`

## Using Images in Your Portfolio

### In the Admin Panel:
1. Go to `/admin`
2. Navigate to the "Projects" tab
3. In the "Images" field, add image paths like:
   ```
   /projects/my-screenshot.png, /projects/another-image.jpg
   ```
4. Save changes

### Image URLs:
- Local images: `/projects/filename.png`
- External images: `https://example.com/image.png`

## Tips
- Keep image sizes reasonable (< 2MB each)
- Use meaningful filenames
- Multiple images per project supported (comma-separated)
- First image shows as the main preview
